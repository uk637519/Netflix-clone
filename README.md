# Netflix Clone

A responsive Netflix Clone built using HTML and CSS. This project recreates the Netflix landing page with a modern user interface and responsive design.

## Features

* Responsive design for mobile, tablet, and desktop
* Netflix-inspired landing page
* Hero section with email signup form
* FAQ section
* Interactive buttons and layout
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3

## Project Purpose

This project was created to improve front-end development skills and gain hands-on experience with responsive web design and modern layouts.
